### Instalar as dependências
```python -m pip install -r requirements.txt```

### Iniciar o servidor
Rodar o arquivo _servidor.py_ utilizando a sua IDE de preferência, ou através do comando:

```python server.py```

Após isso, o backend irá rodar na porta 5000.
